BooKeeping.REST

REST access layer for BooKeeping.Backend.
Author - Rahul Chaudhary



Common cURL Commands:

Command Name cURL Command
User Create 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							    "userId": "john",
							    "firstName": "John",
							    "lastName": "Doe",
							    "primaryEmail": "johndoe@example.com"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/user/create'
User Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/user/info'
Filesystem Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash",
							    "filesystemId": "rash-filesystem-1"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/filesystem/info'
Directory Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash",
							    "filesystemId": "rash-filesystem-1",
							    "filesystemVersion": 0,
							    "directoryPath": "/Music",
							    "directoryName": "Inspirational"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/directory/info'
File Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash",
							    "filesystemId": "rash-filesystem-1",
							    "filesystemVersion": 0,
							    "filePath": "/Music/Inspirational",
							    "fileName": "bourne-theme.mp3"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/file/info'
