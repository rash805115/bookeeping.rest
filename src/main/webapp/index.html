BooKeeping.REST

REST access layer for BooKeeping.Backend.
Author - Rahul Chaudhary



Common cURL Commands:

Command Name cURL Command
User Create 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							    "userId": "john",
							    "firstName": "John",
							    "lastName": "Doe",
							    "primaryEmail": "johndoe@example.com"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/user/create'
User Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/user/info'
Filesystem Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash",
							    "filesystemId": "rash-filesystem-1"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/filesystem/info'
Directory Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash",
							    "filesystemId": "rash-filesystem-1",
							    "filesystemVersion": 0,
							    "directoryPath": "/Music",
							    "directoryName": "Inspirational"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/directory/info'
File Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
								"userId": "rash",
							    "filesystemId": "rash-filesystem-1",
							    "filesystemVersion": 0,
							    "filePath": "/Music/Inspirational",
							    "fileName": "bourne-theme.mp3"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/file/info'
Node Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							  "nodeId": "4"
							}' \
							 'http://localhost:8080/bookeeping.rest/api/node/info'
Node Version Info 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							  "nodeId": "4",
							  "version": 0
							}' \
							 'http://localhost:8080/bookeeping.rest/api/node/version/info'
Commit - Node Version 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							  "commitId": "John Commit 2",
							  "Node_Version": {
							    "nodeId": "n"
							  }
							}' \
							 'http://localhost:8080/bookeeping.rest/api/commit'
Commit - Node Delete 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							  "commitId": "John Commit 7",
							  "Node_Delete": {
							    "nodeId": "d"
							  }
							}' \
							 'http://localhost:8080/bookeeping.rest/api/commit'
Commit - Filesystem Create 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							  "commitId": "John Commit 1",
							  "Filesystem_Create": {
							    "userId": "john",
							    "filesystemId": "john-filesystem-1"
							  }
							}' \
							 'http://localhost:8080/bookeeping.rest/api/commit'
Commit - Filesystem Restore 
							curl -i -X POST \
							   -H "Content-Type:application/json" \
							   -d \
							'{
							  "commitId": "John Commit 10",
							  "Filesystem_Restore": {
							    "userId": "john",
							    "filesystemId": "john-filesystem-1",
							    "nodeIdToBeRestored": "d"
							  }
							}' \
							 'http://localhost:8080/bookeeping.rest/api/commit'
